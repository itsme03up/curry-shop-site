##構成を考える

自分で勝手にFastAPIとReact使って作る
小谷とは「Django + htmx + Alpine.js」構成で

#ディレクトリ構成を作成（モノレポ）
- mkdir -p backend/app/{api,models,core,crud}
- mkdir -p frontend/src/{components,pages,services}
- mkdir -p docker/{backend,frontend}
- mkdir k8s
- touch docker-compose.yml README.md

#FastAPI と React (Vite + TypeScript) を入れる
- backend初期化
cd backend
python3 -m venv venv
source venv/bin/activate
pip install fastapi uvicorn[standard]
deactivate
cd ..
- npm create vite@latest frontend -- --template react-ts
cd frontend
- npm installは自動でやってくれるようになってると思う
cd ..
